Dungeons & Dragons™
Computer Labyrinth Game
Simulator

This game is a simulation of the Dungeons and Dragons™ Computer Labyrinth Game™ made by Mattel Electronics™ released in 1980.

I owned this game back in 1981 when I was 11 years old and only recently rediscovered it so I thought it would be a fun project to recreate it on the internet for others to enjoy.

To play the game simply press the play button below. If you are new to the game I recommend reading the original game manual to get familiar with the game play. The game relies on sound queues for play so be sure to have your speakers turned on or enable the in-game help messages.

Enjoy!
